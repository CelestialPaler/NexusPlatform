# Nexus 使用指南 (User Guide)

> **版本**: 1.0
> **适用版本**: v1.5.0+

## 1. 快速开始 (Quick Start)

### 1.1 启动方式
在 VS Code 中按下 `Ctrl+Shift+P` -> `Run Task`，选择：
*   **`Nexus: 启动开发环境 (Run Dev)`**: 同时启动后端与前端热更新，适合日常使用。
*   **`Nexus: 编译并运行 (Build & Run)`**: 编译为可执行文件并运行，适合验证最终效果。

### 1.2 首次运行检查
确保右下角状态栏显示 "Backend Connected"，且无 "Python Environment Mismatch" 报错。

## 2. 核心功能手册

### 2.1 调试控制台 (Debug Console)
v1.5.0 版本引入了全新的分栏调试界面：
*   **Controls (控件 Tab)**: 测试各类按钮、输入框、状态指示器的交互响应。
*   **Charts (图表 Tab)**: 
    *   **瀑布图 (Waterfall)**: 观察频谱随时间的变化。
    *   **3D 表面图 (Surface3D)**: 查看天线方向图仿真。
    *   **热力图 (Heatmap)**: 分析信道相关性。

### 2.2 网络分析工具
*   **Ping**: 基础连通性测试。
*   **Iperf**: 吞吐量测试。
*   **Wireless Capture**: 调用 Wireshark 核心进行抓包分析。

## 3. 故障排查 (Troubleshooting)

### 3.1 常见问题
*   **Q: 启动后白屏？**
    *   A: 检查前端服务是否启动 (端口 5173)，或查看 `logs/app.log`。
*   **Q: 图表不显示？**
    *   A: 确认 WebGL 是否开启，或显卡驱动是否正常。

### 3.2 不可恢复错误
若遇到环境完全损坏，请运行 Task: `🔧 Reinit Python Environment` 进行重置。
